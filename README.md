# ВКР ООО «КУПИШУЗ» — версия с правками преподавателя

Репозиторий содержит исправленную версию дипломной работы и генератор
`.docx` под требования методических рекомендаций МФЮА.

## Правки, внесённые по замечаниям преподавателя

1. **п. 1.3** — «Зарубежный опыт бизнес-планирования при запуске
   собственного производства в сегменте модной розничной торговли»
   (опыт Inditex / Zara, Fast Retailing / Uniqlo, Nike, Adidas,
   V. F. Corporation / Vans, ASOS, Amazon Essentials, Walmart / George),
   включая сравнительную таблицу 1.2 и семь ключевых принципов, применимых
   для проекта ООО «КУПИШУЗ».
2. **п. 2.3** — «СВОТ- и PEST-анализ внешней и внутренней среды»: вводный
   текст и упоминания согласованы с заголовком (СВОТ-анализ — таблицы 2.6 и
   2.7, PEST-анализ — таблица 2.5).
3. **Глава 3** — «Разработка бизнес-плана по производству собственной
   торговой марки обуви ООО «КУПИШУЗ»», структурирована в два параграфа:
   - **3.1 «Мероприятия по совершенствованию системы бизнес-планирования и
     запуску собственного производства»** — концепция, маркетинг,
     производство, организация, финансовый план.
   - **3.2 «Оценка экономической эффективности предложенных мероприятий и
     анализ рисков проекта»** — с пересогласованными внутренними ссылками.
4. **Оглавление, введение (задачи и раздел «Структура работы») и заключение**
   синхронизированы с новой структурой; согласованы ссылки на приложения
   А, Б, В, Г.

## Оформление

Документ собран строго по разделу 8 методических рекомендаций МФЮА:

- Times New Roman, 14 пт; межстрочный 1,5; абзацный отступ 1,25 см;
- Поля 30 / 10 / 20 / 20 мм (левое / правое / верхнее / нижнее);
- Выравнивание текста по ширине;
- Нумерация страниц снизу по центру арабскими цифрами, на титульном листе номер не печатается;
- Шрифт внутри таблиц 12 пт, межстрочный 1,0; подписи таблиц слева, рисунков по центру;
- Список источников разбит на разделы (нормативные акты, литература,
  периодические издания, электронные ресурсы, ГОСТы, иные источники).

## Файлы

- `output/VKR_KupiShuz_revised.docx` — итоговый файл диплома;
- `output/VKR_KupiShuz_revised.pdf` — превью PDF, собранное через LibreOffice;
- `vkr_content.py` — структурированное содержание ВКР;
- `docx_styles.py` — стилистическое оформление по методичке;
- `build_vkr.py` — сборщик `.docx`.

## Как пересобрать .docx

```bash
pip install python-docx pymupdf
python3 build_vkr.py
```

После открытия `.docx` в Microsoft Word рекомендуется обновить оглавление
(`Ctrl + A → F9`) — поле `{TOC}` подтянет актуальные номера страниц.
